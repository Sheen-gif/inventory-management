Request Data Flow
1
User changes a filter
FilterBar.vue updates a reactive ref in useFilters.js — e.g. selectedLocation, selectedCategory, selectedPeriod, or selectedStatus.
2
Watch triggers in the active View
Each view (Orders, Inventory, Dashboard, etc.) watches the filter composable and calls getCurrentFilters() to build a params object.
3
api.js constructs the HTTP request
axios sends a GET with URLSearchParams — e.g. GET /api/orders?warehouse=Tokyo&status=Shipped&month=Q1-2025
4
FastAPI filters in-memory data
main.py applies sequential filters: warehouse → category → status → month/quarter. Returns a Pydantic-validated JSON array.
5
Vue stores and derives display data
Raw response stored in a ref(). Computed properties derive filtered/sorted/aggregated values. Template re-renders reactively.
